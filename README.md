# Lutro Brawler

A **Street Fighter** clone made in **Lua**, adapted to run on both [Love2D](https://love2d.org) and [Lutro](http://lutro.libretro.com).

![Gameplay Screenshot](https://github.com/user-attachments/assets/d5fee812-c1bf-459a-9e25-034702be62af)


## Controls

### Player 1 (P1)
| Action   | Key(s)  |
|----------|---------|
| Move     | W A S D |
| Action 1 | R       |
| Action 2 | T       |

### Player 2 (P2)
| Action   | Key(s)                      |
|----------|-----------------------------|
| Move     | Arrow Keys (↑ ↓ ← →)        |
| Action 1 | Numpad 1                    |
| Action 2 | Numpad 2                    |

## Running the Game

**With Love2D:**
```shell
make run/love
```

**With Lutro:**
```shell
make run/lutro
```

If needed, download the **lutro** core:
```shell
make get/lutro-core
```
